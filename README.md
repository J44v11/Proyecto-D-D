# Proyecto-D&D

Este es un pequeño proyecto desarrollado en Java para la clase de programación. Básicamente, es un simulador muy sencillo de personajes de un videojuego. 

El objetivo principal de la práctica es aplicar los conceptos de Programación Orientada a Objetos (POO), trabajando con una clase base y utilizando la herencia para crear tipos específicos de luchadores, como un Guerrero y un Mago.

Además, este repositorio también me sirve como entorno de pruebas para aprender a manejar Git correctamente, haciendo commits y moviéndome entre distintas ramas.
